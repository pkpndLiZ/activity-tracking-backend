# Activity tracking Backend

## วิธี setup สำหรับ clone ครั้งแรก

- run คำสั่งด้านล่าง เพื่อสร้าง file .env

```bash
npm run setup
```

- run คำสั่ง เพื่อ install dependencies

```bash
npm install
```

## วิธีการ run server

- run คำสั่ง `npm run dev`
